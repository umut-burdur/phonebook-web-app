# Phonebook Application (Python Flask) deployed on AWS Application Load Balancer with Auto Scaling and Relational Database Service

## Description

The Phonebook Application aims to create a phonebook application in Python and deployed as a web application with Flask on AWS Application Load Balancer with Auto Scaling Group of Elastic Compute Cloud (EC2) Instances and Relational Database Service (RDS).

There are two folders for this project. One of them is using the AWS CloudFormation service, and the other is using Terraform for deploying the project. There are some fields that needed to be changed; please look out for comments.
